# Math

## API

```js
let a = 0b010101;  //--> 21  二进制写法
let b = 0o666  //--> 438  八进制写法

Math.trunc(3.14);  //--> 3  截断，只保留整数
Math.sign(-12);  //--> -1  判断正数（1）、负数（-1）、0
Math.cbrt(27);  //--> 3  立方根
Math.pow(2, 3);  //--> 8  计算次方 == (2 ** 3)
Math.sqrt(4);  //--> 2  开方

let arr = [1, 2, 3, 4, 5]
Math.max(...arr);  //--> 5  获取数组最大值
Math.min(...arr);  //--> 1  获取数组最小值
```
