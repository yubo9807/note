# 循环语句

## for

```go
package main

func main() {
    out: for i := 0; i < 5; i++ {
		// code...
		continue  // 跳过循环
		for j := 0; j < 5; j++ {
			break out  // 终止标记循环
		}
	}

	var j = 0
	for j < 5 {
		// code...
			j++
	}

    // 死循环
    for {
        // code...
    }
}
```

### goto

```go
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
		goto log
	}

	return

	log: fmt.Println(111)
}
```
