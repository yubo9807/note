# Error

- 错误：可能哪些地方出现了问题，在意料之中；
- 异常：不该出现错误的地方出现了问题，在意料之外。

## 错误

```go
package main

import (
		"errors"
		"fmt"
)

func main() {
    err := errors.New("error")
		fmt.Println(err)  //--> error
		fmt.Printf("%T\n", err)  //--> *errors.errorString
}
```

## 异常

```go
package main

import "fmt"

func main() {
    defer func() {
				if msg := recover(); msg != nil {
				    fmt.Println(msg)
				}
		}()

    fmt.Println(1)
    panic("程序异常")
		fmt.Println(2)
}

```
