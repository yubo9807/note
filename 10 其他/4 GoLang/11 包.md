# 包

- 内置函数：init main 自执行函数
    - init 在 main 函数之前执行；
    - init 可定义多个，main 函数只能在 main 包中调用，并且只能定义一个；
- init 执行顺序：
    - 同一个 package 下的文件，以字符串的形式排序，挨个执行当中的 init 函数；
    - 不同 package，如果不相互依赖的话，按 import 顺序调用包中的 init 函数；
    - package 存在依赖关系，调用顺序按最后被依赖的最先被初始化；
    - 被多个包导入，只会初始化一次；

```
├─ bin
├─ pkg
├─ src
└─ main.go
```
