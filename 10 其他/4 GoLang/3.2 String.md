# 字符串类型

## 编码问题

- ASCII（美国标准信息交换码）
- gbk（中国编码表，兼容了 ASCII）
- Unicode（号称统一了全世界，并没有兼容 gbk，而是把汉字重新编排了）
    - UTF-8，UTF-16，UTF-32...

## '' 与 "" 的区别

```go
package main

import "fmt"

func main() {
    var (
        a = 'A'
        b = "A"
    )
    fmt.Printf("%T, %d\n", a, a)  //--> int32, 65  '' 会将数字转为 UTF-8
    fmt.Printf("%T, %s\n", b, b)  //--> string, A
}
```
