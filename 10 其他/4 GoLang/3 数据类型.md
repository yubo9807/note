# 数据类型

## 基本数据类型

- 布尔类型：bool
- 数值类型：int float complex
- 字符串：string

## 复合数据类型（派生类型）

- array slice map function pointer struct interface channel

## 类型转换

> 语法格式：`Type(Value)`

```go
package main

import "fmt"

func main() {
    var a int8 = 10
		var b = int16(a)
		fmt.Println(a, b)  //--> 10 10

		var c = 3.14
		var d = int(c)  //--> 向下去整
		fmt.Println(c, d)  //--> 3.14 3

		var f = float32(a)
		fmt.Printf("%T, %f\n", f, f)  //--> float32, 10.000000
}
```
